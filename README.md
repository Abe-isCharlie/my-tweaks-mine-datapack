# My Tweaks

## Description

It's just a datapack with some tweaks and more debris. Nothing too fancy, just some random ideas.

## Content

Currently adds/changes:

- Increases the generation of Anciet Debris
- Ancient Debris generates in Soul Sand Valleys
- It is possible to get Quartz back from Quartz Blocks via crafting table (Integration with Crushing Wheel from [Create mod](https://www.curseforge.com/minecraft/mc-mods/create)).
- More soon...
